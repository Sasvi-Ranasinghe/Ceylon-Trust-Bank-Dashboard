# 🏦 BankInsight

## Banking Customer Analytics Dashboard

BankInsight is a Banking Customer Analytics Dashboard developed using Python.

This project is designed to help bank managers and business analysts analyze banking data through interactive dashboards.

---

## Project Objectives

- Analyze customer information
- Analyze account performance
- Analyze loan performance
- Analyze transactions
- Analyze branch performance
- Generate business insights

---

## Technologies

- Python
- Pandas
- NumPy
- Plotly
- Dash
- Matplotlib
- Seaborn
